OPTIONAL

Create home screen shortcuts

⌃

Open apps distraction-free in Safari with one tap.

Example for Instagram:

  1. Open the Shortcuts app on your iPhone
  2. Tap + to create a new shortcut
  3. Tap the name of your new shortcut (in the top) and rename it to e.g., "Instagram Safari"
  4. Search for the "Open URLs" action and tap it
  5. Tap URL and input "instagram.com"
  6. Tap the 'Share' icon in the bottom right and select 'Add to Home Screen'
  7. Select the icon you want and tap 'Add'

This extension lets you hide distractions on any website, with pre-configured options for YouTube, Facebook, Twitter, Instagram, Linkedin, and Google.com. It's open source and collects zero data - you can find the source code on GitHub.

To turn it on, first toggle it on in Safari Extensions preferences. Then go to a website, click the extension icon in the toolbar, and allow access.
